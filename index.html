Flexbox

Flexbox digunakan untuk layout satu dimensi (baris atau kolom). Flexbox selalu punya 2 sumbu (axis):

  1. Main Axis Arah flex-direction, atas dan bawah, justify-content mengatur main axis.
  2. Cross Axis Arah yang tegak lurus dengan main axis, kiri dan kanan, align-items mengatur cross axis

Grid

Grid digunakan untuk layout dua dimensi (baris dan kolom).

  1. Grid = GARIS
  2. Row/Column adalah jarak antar garis
  3. start & end bukan arah mereka hanya penanda batas
