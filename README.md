# Flex & Grid Playground

Flex & Grid Playground adalah sebuah **web interaktif** untuk membantu memahami cara kerja **CSS Flexbox** dan **CSS Grid** secara visual dan langsung (live preview).  
Proyek ini dibuat sebagai **media belajar** untuk melihat hubungan antara properti CSS dan perubahan layout secara real-time.

- pengguna bisa **mencoba properti CSS tanpa menulis kode**
- perubahan langsung terlihat di **Live Preview**
- konsep layout menjadi lebih **visual dan intuitif**
